M

Memory Function Test Guide

  1. Enter a number (e.g., 5)
  2. Press M+ to add to memory (M indicator appears)
  3. Enter another number (e.g., 3)
  4. Press M+ again to add to memory
  5. Clear the display with AC
  6. Press MR to recall memory (should show 8)
  7. Press MC to clear memory (M indicator disappears)
